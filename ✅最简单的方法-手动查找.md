# ✅ 最简单的方法 - 手动查找

## 💡 手动查找 Git 仓库（最简单）

不需要复杂命令，按照以下步骤操作：

---

## 📋 步骤 1：打开文件资源管理器

1. 按 `Win + E` 打开文件资源管理器
2. 或者点击任务栏的文件资源管理器图标

---

## 📋 步骤 2：进入桌面文件夹

1. 点击左侧的"**桌面**"文件夹
2. 或者直接进入：`C:\Users\26872\Desktop`

---

## 📋 步骤 3：查找可能的项目文件夹

**查看桌面上是否有以下文件夹：**

- `mindcube` ← 最可能的
- 或者包含 `心理网站` 的文件夹
- 或者其他项目文件夹

---

## 📋 步骤 4：检查是否有 `.git` 文件夹

**对每个可能的文件夹：**

1. **打开文件夹**
2. **查看是否有 `.git` 文件夹**
   - 如果看不到，需要显示隐藏文件：
     - 点击文件资源管理器的"**查看**"标签
     - 勾选"**隐藏的项目**"复选框
3. **如果看到 `.git` 文件夹，那就是 Git 仓库！**

---

## 📋 步骤 5：告诉我路径

**一旦找到包含 `.git` 的文件夹，告诉我：**
- 文件夹的完整路径
- 例如：`C:\Users\26872\Desktop\mindcube`

---

## 🚀 或者运行搜索脚本

**我已经创建了一个搜索脚本，你可以运行：**

```powershell
powershell -ExecutionPolicy Bypass -File search-git.ps1
```

这会自动搜索并显示所有找到的 Git 仓库。

2. 或者你的项目文件夹叫什么名字？
3. 或者你之前在哪个目录执行的 `git push`？
