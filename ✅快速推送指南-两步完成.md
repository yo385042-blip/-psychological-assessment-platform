   ```powershell

   ```

   ```powershell
   .\push-code.ps1
   # 或者提供完整路径
   ```

### 方式 B：手动执行命令

在 Git 仓库目录中执行：

```powershell

git add functions/_middleware.js

# 2. 提交
git commit -m "简化 Functions middleware：直接处理所有路由，返回 index.html"

# 3. 推送
git push origin main
```

## 💡 快速检查方法

**在 PowerShell 中执行这个命令，看看能找到哪些包含 .git 的文件夹：**

```powershell
Get-ChildItem -Path "$env:USERPROFILE\Desktop" -Directory -Recurse -Depth 2 -Filter ".git" -ErrorAction SilentlyContinue | Select-Object -First 5 @{Name="Git目录";Expression={$_.Parent.FullName}}
```

这会列出桌面上所有包含 `.git` 的文件夹。
