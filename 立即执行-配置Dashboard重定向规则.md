# 🚀 立即执行：配置 Dashboard 重定向规则

## ❌ 问题确认

- ✅ `_redirects` 文件上传了
- ❌ 部署后文件消失
- ❌ 网站无法访问

**原因：** 直接上传 dist 时，Cloudflare Pages 不会处理 `_redirects` 文件。

---

## ✅ 解决方案：在 Dashboard 手动配置（最可靠）

### 📋 步骤 1：进入项目

1. 登录 Cloudflare Dashboard
2. Workers & Pages → 你的项目

### 📋 步骤 2：找到 Redirects 标签

在项目页面顶部，你会看到标签：

- 资产已上传
- 函数
- **重定向** ← **点击这个！**
- 标头

### 📋 步骤 3：添加规则

1. 点击 **"添加规则"** 或 **"Create rule"**

2. 填写：
   ```
   匹配模式：/*
   重写为：/index.html
   状态码：200
   ```

3. 点击保存

### 📋 步骤 4：测试

等待几秒，然后访问网站测试。
