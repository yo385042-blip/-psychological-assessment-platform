# 修复导入错误说明

## 问题
`QuestionImport.tsx` 中导入了以下函数，但这些函数在 `questionImport.ts` 中不存在：
- `parseExcel`
- `parseMarkdown`
- `parseXML`
- `parseHTML`

## 解决方案
已在 `src/utils/questionImport.ts` 中添加了所有缺失函数的实现：

### 1. `parseExcel(file: File, alias: Record<string, string>): Promise<ParseResult>`
- 解析 Excel 文件（.xlsx, .xls）
- 目前实现：尝试将文件作为文本读取，如果是二进制格式会提示用户转换为 CSV
- 注意：完整的 Excel 解析需要引入 `xlsx` 库，当前为简化实现

### 2. `parseMarkdown(text: string): ParseResult`
- 解析 Markdown 格式的题目文件（.md, .markdown）
- 支持格式：
  - `## 题目标题`
  - `- 选项A`
  - `答案：A`

### 3. `parseXML(text: string): ParseResult`
- 解析 XML 格式的题目文件（.xml）
- 使用 `DOMParser` 解析 XML
- 支持 `<question>` 标签结构

### 4. `parseHTML(text: string): ParseResult`
- 解析 HTML 格式的题目文件（.html, .htm）
- 从 HTML 页面中提取题目信息
- 支持多种 HTML 结构：
  - 表单（`<form>`）
  - 列表（`<ul>`, `<ol>`）
  - 题目容器（`.question`, `.quiz-item` 等）
  - 通用标题结构（`<h1>`-`<h6>`）

## 辅助函数
还添加了以下辅助函数：
- `extractQuestionFromForm()` - 从表单中提取题目
- `extractQuestionFromList()` - 从列表中提取题目
- `extractQuestionFromContainer()` - 从容器中提取题目
- `extractQuestionsGeneric()` - 通用题目提取
